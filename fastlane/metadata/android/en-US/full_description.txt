<p><i>c:geo</i> is an open source, full-featured, always ready-to-go unofficial client for geocaching.com and offers basic support for other geocaching platforms (such as Opencaching). It does not require a web browser or exports - just install it and start right away.</p><p><br><b>Main features:</b></p><ul><li>View caches on a Live Map</li><li>Use Google Maps or OpenStreetMap</li><li>Search for caches by various criteria</li><li>Log your finds online or offline</li><li>Store cache information on your device</li><li>Create and manage waypoints</li><li>Navigate using compass, map, or other apps</li><li>Import/Export GPX files</li><li>Full support for trackables</li><li>Offline caching functions including offline maps</li></ul><p><i>c:geo</i> is a simple to use but powerful geocaching client with a lot of additional features. All you need to get started is an existing account on geocaching.com or another geocaching platform (like Opencaching). Find caches using the live map or by using one of the many search functions.</p><p>Navigate to a cache or a waypoint of a cache with the built-in compass function, the map or hand over the coordinates to various external apps (e.g. Radar, Google Navigation, StreetView, Locus, Navigon, Sygic and many more).</p><p>Store cache information to your device directly from geocaching.com as well as via GPX file import to have it available whenever you want. You can manage your stored caches in different lists and can sort and filter them according to your needs. Stored caches together with offline map files or static maps can be used to find caches without an internet connection (e.g. when roaming). Logs can be posted online or stored offline for later submission or exported via field notes.</p><p>Search and discover trackables, manage your trackable inventory and drop a Trackable while posting a cache log.</p><p>If you have problems installing or using c:geo please first have a look at <a href='https://faq.cgeo.org' target='_blank' rel='nofollow noopener'>our FAQ</a> or consult <a href='https://manual.cgeo.org' rel='nofollow noopener'>the user guide</a>. If there are still problems, contact support via email.</p><p>If you would like to know why <i>c:geo</i> needs the permissions requested, please <a href='https://www.cgeo.org/faq#permissions' rel='nofollow noopener'>check here</a> for an explanation.</p>
