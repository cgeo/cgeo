### Важные примечания к релизу

**Вне рамок**

В соответствии с политикой Play Store мы обновили уровень Android API в этой версии c: geo + мы изменили некоторые процедуры компоновки экрана. Это может привести к некоторым нежелательным побочным эффектам, особенно в новых версиях Android. Если у вас возникнут какие-либо проблемы с этой версией c:geo, пожалуйста, сообщите об этом на [GitHub](https://github.com/cgeo/cgeo) или по электронной почте [support@cgeo.org](mailto:support@cgeo.org)

**Устаревшие карты**

Как было объявлено в релизах 2025.07.17 и 2025.12.01, мы наконец-то удалили устаревшие версии наших карт. Вы будете автоматически переведены на нашу новую ЕдинуюКарту и не заметите никаких отличий, за исключением нескольких новых функций, некоторые из которых
- Поворот карты для карт на основе OpenStreetMap (онлайн + офлайн)
- Всплывающее окно кластера для Google Maps
- Возможность убрать ненужные вам источники карт
- Диаграмма высот для маршрутов и треков
- Переключение между списками непосредственно с карты
- "Режим вождения" для карт на основе OpenStreetMap
- Длительное нажатие на трек/индивидуальный маршрут открывает дополнительные возможности

- Новое: При оптимизации маршрута кэшируются данные расчетов
- Новое: При включении режима реального времени путевые точки текущей установленной цели остаются видимыми
- Новое: Длительное нажатие на навигационную линию открывает карту высот (ЕдинаяКарта)
- Новое: Показать сгенерированные точки на карте
- Новое: Загрузка тайников упорядочена по расстоянию
- Исправлено: Дублирование отдельных пунктов маршрута
- Новое: Поддержка темы Motorider (только VTM)
- Новое: Не отображать карту в разделе карт. Будут показаны только тайники и т. п.
- Изменение: Максимальное расстояние для соединения точек на историческом треке уменьшено до 500 м (можно настроить)
- Новое: Возможность импортировать файлы KML в виде треков (например, отслеживаемый маршрут)

- Новое: Информационное поле для диаграммы высот, показывающее оставшееся расстояние, подъем, спуск

- Исправлено: При быстрой настройке карты могли отображаться кнопки "1"/"2" для пустых профилей маршрутов после переключения языка
- Новое: Вычисление недостающих данных о высотах при импорте треков (если данные о высотах загружены)
- Исправлено: Загрузка плитки карты останавливалась при определенных условиях (только для онлайн-карт OpenStreetMap)
- Новое: Символы тайников на основе условий
