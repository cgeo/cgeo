c:geoはオープンソース、フル機能、いつでも出かけられるgeocachig.com用の非公式クライアントです。Opencachingなどの別のジオキャッシングプラットフォームのための基本的な機能もあります。 他にウェブブラウザやアプリは必要ありません。インストールするだけですぐに使用できます。

主な機能:
- ライブマップでキャッシュを表示
- GoogleマップまたはOpenStreetMapsを使用する
- 様々な条件でキャッシュを検索
- オンラインまたはオフラインでログを記録する
- 端末にキャッシュ情報を保存する
- ウェイポイントの作成と管理
- コンパス、マップ、またはその他のアプリを使用して案内
- GPXファイルのインポート/エクスポート
- トラッカブルのフルサポート
- オフライン地図を含むオフラインキャッシュ機能

c:geoは簡単に使用でき、多くの機能を備えたパワフルなジオキャッシングクライアントです。 geocaching.comや他のジオキャッシングプラットフォーム(Opencachingなど)のアカウントがあればすぐに始まられます。
ライブマップや充実した検索機能でキャッシュを探せます。

キャッシュやウェイポイントへ内臓コンパスや地図でナビすることができます。また、外部アプリと連携してナビすることもできます(例:Radar、Googleマップナビ、Googleマップのストリートビュー、Locus、Navigon、Sygicなどなど)。

geocaching.comからキャッシュ情報をデバイスに直接保存することができ、GPXファイルをインポートすることも可能です。
保存したキャッシュ情報はリストで分類して管理したり、ソートやフィルター機能もあります。
オフライン地図を使ったりオフライン用に地図を保存することもできるのでインターネット接続なしにキャッシュを探せます。 (ローミング時などに有用です)
ログはオンラインで投稿することも、後で投稿するためにオフラインで保存することも、フィールドノートとしてエクスポートすることもできます。

トラッカブルアイテムを検索したり、ご自身のトラッカブルアイテムを管理したり、トラッカブルアイテムのログを投稿することもできます。

インストール時や使っていて問題があったら最初にFAQ (https://faq.cgeo.org)やユーザガイド(https://manual.cgeo.org)(英語)をご覧ください。
それでも問題が解決しなければサポートにメールで連絡してください(英語、ドイツ語、フランス語かスロベニア語でお願いします)。

c:geoが要求する権限についてはhttps://www.cgeo.orgでご確認ください。
