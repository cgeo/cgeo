c:geo是geocaching.com的一个非官方客户端，它不仅开源、全能、随时可用，还支持其它geocaching平台（例如Opencaching）。 它不需要依赖web浏览器或导出记录—只需要安装并启动它即可。

主要功能：
- 在实时地图上查看藏点
- Use Google Maps or OpenStreetMap
- 使用各种条件搜索藏点
- 在线或离线记录你找到的藏点
- 将藏点信息保存到设备上
- 创建并管理路径点
- 使用指南针、地图或其它应用进行导航
- 导入/导出 GPX 文件
- 对可跟踪物的完全支持
- 包括离线地图在内的离线寻宝功能

c:geo易用且强大，具有很多额外的功能。 只需要一个geocaching.com或其它geocaching平台（比如Opencaching）的账号，你就可以开始玩耍了。
你可以使用实时地图或其它搜索功能来寻找藏点。

你可以使用内置指南针、地图或其他外部应用（Radar、Google Navigation、StreetView、Locus、Navigon、Sygic等）导航去藏点或路径点。

除了直接将藏点信息从geocaching.com保存到你的设备上外，你也可以随时通过导入GPX文件来使藏点信息生效。
你可以管理不同列表中保存的藏点，按需进行排序及过滤。
同时保存藏点和离线地图文件（或静态地图）可以让你在没有网络连接的情况下也能进行寻宝活动（例如： 漫游时）。
日志可以直接在线发布，也可以为稍后发布而离线保存，还可以通过笔记导出。

搜索并发现可追踪物，管理你的可追踪物仓库以及在发布藏点日志时放下它。

如果你在安装或使用c:geo的过程中遇到了问题，请先看一下我们的FAQ（https://faq.cgeo.org）或者查阅用户指南（https://manual.cgeo.org）。
如果以上两个都没有解决你的问题，那就通过电子邮件联系支持吧。

如果您想知道为什么 c:geo 需要所要求的权限，请查看 https://www.cgeo.org/faq#permissions 获取解释。
