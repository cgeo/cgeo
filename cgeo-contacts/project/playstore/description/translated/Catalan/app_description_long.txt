Es tracta d’un plugin de contactes per a c: geo, una aplicació de geocatxing de codi obert. Només funcionarà amb l'aplicació c: geo instal·lada i permet l'accés directe a la vostra agenda quan feu clic a un nom d'usuari a c: geo.

Ús:
Quan feu clic a un nom d’usuari a c: geo (ex,,. al registre d'un catxé) el menú contindrà un element per obrir la targeta de contacte corresponent als contactes del telèfon. Això us permetrà trucar directament o enviar missatges a l’usuari.

Observacions:
El nom d’usuari de la persona s’ha d’emmagatzemar al camp de nom o àlies dels vostres contactes perquè el connector coincideixi amb el nom d’usuari de geocatxing amb el contacte.
