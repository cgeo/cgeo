Это плагин контактов для c:geo — геокэшинг-приложения с открытым исходным кодом. Плагин не работает самостоятельно, ему требуется установленное c:geo. Этот плагин позволит получить доступ к телефонной книге при нажатии на имя пользователя в c:geo.

Использование:
При нажатии на имя пользователя в c:geo (напр. в блокноте тайника) в меню появится пункт для открытия соответствующей контактной карточки в контактах вашего телефона. Это позволит вам напрямую позвонить или отправить сообщение пользователю.

Замечание:
Ник пользователя должен быть сохранен в поле имени или псевдонима в вашем списке контактов, чтобы плагин смог найти этого пользователя по нику.
