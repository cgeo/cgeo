c:geo는 geocaching.com의 오픈 소스이며 모든 기능을 갖춘 항상 준비된 비공식 클라이언트이며 다른 지오캐싱 플랫폼 (예: Opencaching)에 대한 기본 지원을 제공합니다. 웹 브라우저나 내보내기가 필요하지 않습니다. 그냥 설치하고 바로 시작하십시오.

주요 특징:
- 실황 지도에서 캐시 보기
- 구글 지도 또는 OpenStreetMap 사용
-다양한 기준으로 캐시 검색
-찾기를 온라인 또는 오프라인으로 로그하기
- 전기기구에 캐시 정보 저장
- 지점 생성 및 관리
- 나침반, 지도 또는 다른 앱을 사용하여 탐색
- GPX 파일 가져오기/내보내기
-추적 아이템을 완벽하게 지원
- 오프라인 지도를 포함한 오프라인 캐싱 기능

c:geo는 사용하기 간단하지만 많은 추가 기능이 있는 강력한 지오캐싱 클라이언트입니다. 시작하려면 geocaching.com의 기존 계정이나 다른 지역 캐싱 플랫폼 (예 Opencaching)만 있으면됩니다.
실황 지도를 사용하거나 많은 검색 기능 중 하나를 사용하여 캐시를 찾으십시오.

내장 된 나침반 기능을 사용하여 캐시 또는 캐시의 지점으로 이동하거나 지도를 다양한 외부 앱 (예 Radar, Google Navigation, StreetView, Locus, Navigon, Sygic 등)으로 좌표를 넘겨줍니다.

캐시 정보를 geocaching.com에서 GPX 파일 가져 오기를 통해 장치에 직접 저장하여 필요할 때 언제든지 사용할 수 있습니다.
저장된 캐시를 다른 목록으로 관리하고 필요에 따라 정렬하고 필터링 할 수 있습니다.
오프라인 지도 파일 또는 정적지도와 함께 저장된 캐시를 사용하면 인터넷에 연결되지 않은 캐시를 찾을 수 있습니다 (예  로밍 중일 때)
나중에 제출하기 위해 로그를 온라인으로 게시하거나 오프라인으로 저장하거나 필드 메모를 통해 내보낼 수 있습니다.

캐시 로그를 게시하는 동안 추적 아이템 항목을 검색 및 검색하고, 추적 아이템 인벤토리를 관리하고 추적 아이템 항목을 삭제하십시오.

c:geo 설치 또는 사용에 문제가있는 경우 먼저 FAQ (https://faq.cgeo.org)를 보거나 사용 설명서 (https://manual.cgeo.org)를 참조하십시오.
여전히 문제가 있으면 이메일을 통해 지원 부서에 문의하십시오.

c:geo에 요청 된 권한이 필요한 이유를 알고 싶다면 https://www.cgeo.org/faq#permissions에서 설명을 확인하십시오.
